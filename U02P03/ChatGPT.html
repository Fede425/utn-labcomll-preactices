ChatGPT

¡Perfecto! Para lograr eso con Flexbox, primero necesitarás un contenedor para tus cajas laterales y luego aplicar Flexbox para distribuirlas horizontalmente. Aquí hay un ejemplo básico para empezar:

HTML:

html
<div class="container">
  <div class="box">
    <h2>Título 1</h2>
    <p>Texto de la caja 1.</p>
  </div>
  <div class="box">
    <h2>Título 2</h2>
    <p>Texto de la caja 2.</p>
  </div>
</div>

CSS:

css
.container {
  display: flex; /* Establece el contenedor como un contenedor flex */
}

.box {
  flex: 1; /* Hace que las cajas tengan el mismo tamaño */
  border: 1px solid #000; /* Añade bordes para que puedas verlas */
  padding: 10px; /* Agrega espacio interior para que no estén pegadas a los bordes */
  margin: 10px; /* Agrega margen entre las cajas */
}

Con este código, las cajas se colocarán una al lado de la otra horizontalmente dentro del contenedor. Puedes ajustar los tamaños y estilos según tus necesidades. ¿Hay algo más en lo que necesites ayuda?

Si deseas que el color de fondo del título (h2) ocupe todo el ancho del contenedor (div.box), necesitas establecer el color de fondo en el h2 y ajustar su margen para que coincida con el borde del contenedor. Aquí te dejo cómo hacerlo:

HTML:

html
<div class="container">
  <div class="box">
    <h2>Título 1</h2>
    <p>Texto de la caja 1.</p>
  </div>
  <div class="box">
    <h2>Título 2</h2>
    <p>Texto de la caja 2.</p>
  </div>
</div>

CSS:

css
.container {
  display: flex;
}

.box {
  flex: 1;
  border: 1px solid #000;
  padding: 10px;
  margin: 10px;
}

.box h2 {
  background-color: lightblue; /* Cambia el color de fondo del título */
  margin: -10px -10px 10px -10px; /* Ajusta el margen para que coincida con el borde del contenedor */
  padding: 10px; /* Agrega espacio interior para que el texto no esté pegado al borde del título */
}

Con esto, el color de fondo del h2 ocupará todo el ancho del contenedor div.box, ya que hemos ajustado el margen del h2 para que coincida con los bordes del contenedor. Puedes personalizar el color de fondo y otros estilos según tus preferencias. ¿Hay algo más en lo que necesites ayuda?
